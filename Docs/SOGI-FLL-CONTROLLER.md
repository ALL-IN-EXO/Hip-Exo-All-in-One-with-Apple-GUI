# SOGI-FLL 算法详解：实时生成与髋关节角速度同相的正弦力矩

## 1. 问题背景

### 1.1 目标

髋关节外骨骼要做正功，需要力矩 τ_exo 与关节角速度 ω 同号：

$$P(t) = \tau_{\text{exo}}(t) \cdot \omega(t)$$

- P > 0：外骨骼在**助力**（做正功）
- P < 0：外骨骼在**阻碍**（做负功/抽能）

如果 τ_exo 的波形是一个与 ω **完全同相的正弦**，那么 τ·ω 在整个步态周期内都 ≥ 0，正功最大化。

### 1.2 难点

直接把 τ_exo = k·ω 当然同号，但这是**负阻尼**，增益一大就不稳定。我们真正想要的是：

> **用 ω 的相位信息，生成一个干净的、同相的、幅值可控的正弦信号** —— 把 ω 当"节拍器"，不把它当"幅值模板"。

这就需要从嘈杂的实时 ω 信号中**在线提取瞬时相位 φ(t)**。

---

## 2. 算法选型：为什么选 SOGI-FLL

| 方案 | 延迟 | 频率自适应 | 抗噪 | 复杂度 | 缺点 |
|---|---|---|---|---|---|
| 过零检测 + 周期插值 | 极小 | 有（但抖） | 差 | O(1) | 噪声误触发、首周期瞎 |
| 滑窗 Hilbert | 半窗长 | 否 | 中 | O(N) | 延迟大，实时不友好 |
| **SOGI-FLL** | **~1/(k·f0)** | **是** | **好（带通）** | **O(1)** | **启动需 1-2 周期** |
| 自适应振荡器 AFO | 小 | 是 | 好 | O(1) | 两个学习率要调 |

SOGI-FLL 来自电力电子领域（Rodriguez et al. 2006），本质是一个**自带正交信号发生器的窄带通滤波器 + 频率锁定环**，一步完成：去噪、提取基频、生成同相/正交分量、跟踪频率变化。

---

## 3. 数学原理

### 3.1 SOGI 核心：二阶广义积分器

输入：原始角速度 ω(t)（标量，deg/s）

状态变量：
- **x1(t)**：同相分量（稳态下 ≈ A·sin(φ)，与 ω 同相）
- **x2(t)**：正交分量（稳态下 ≈ -A·cos(φ)，滞后 ω 90°）

微分方程：

```
ε(t) = ω(t) - x1(t)                     ← 误差 = 输入 - 同相估计

dx1/dt = k · ωn · ε  -  ωn · x2         ← 同相分量更新
dx2/dt = ωn · x1                         ← 正交分量更新
```

其中：
- **ωn = 2π·f0**：中心角频率（后面由 FLL 自适应更新）
- **k ≈ 1.41**：阻尼系数（√2 对应临界阻尼附近，平衡锁定速度和抗噪）

### 3.2 为什么 x1 与 ω 同相？—— 传递函数分析

对上述方程做 Laplace 变换，可以得到从 ω 到 x1、x2 的传递函数：

```
H_x1(s) = X1(s)/Ω(s) = (k·ωn·s) / (s² + k·ωn·s + ωn²)     带通
H_x2(s) = X2(s)/Ω(s) = (k·ωn²)  / (s² + k·ωn·s + ωn²)     低通
```

在 s = j·ωn（恰好在中心频率上）：

```
H_x1(j·ωn) = (k·ωn · j·ωn) / (-ωn² + j·k·ωn² + ωn²)
            = j·k·ωn² / (j·k·ωn²)
            = 1                                  ← 增益=1，相移=0°

H_x2(j·ωn) = k·ωn² / (j·k·ωn²)
            = 1/j = -j                           ← 增益=1，相移=-90°
```

**所以在 ωn 处，x1 完美跟踪 ω（零相移），x2 滞后 90°。**

对于偏离 ωn 的频率成分，增益快速衰减（带通特性）→ 自动去噪。

### 3.3 瞬时相位和幅值提取

稳态下，如果 ω ≈ A·sin(ωn·t)，那么：
- x1 ≈ A·sin(ωn·t)
- x2 ≈ -A·cos(ωn·t)

这构成一个**解析信号对**，用 atan2 可以还原出瞬时相位：

```
φ(t) = atan2(x1, -x2) = atan2(A·sin(ωn·t), A·cos(ωn·t)) = ωn·t
```

此时：

```
sin(φ) = sin(ωn·t)  与 ω 同相 ✓
```

瞬时幅值：

```
amp(t) = √(x1² + x2²) ≈ A
```

> **关键点：`atan2(x1, -x2)` 的参数顺序不能反。**
> 如果错写成 `atan2(-x2, x1)`，输出的 sin(φ) 会变成 cos(ωn·t)，与 ω **正交**而非同相，
> 相当于白白引入 90° 相位偏差。

### 3.4 FLL：频率锁定环

如果输入频率和初始 ωn 不一致，SOGI 输出的 x1 和 x2 会有系统性误差。FLL 用这个误差信号来在线调节 ωn：

```
dωn/dt = -γ · ε · x2 / (x1² + x2² + ε₀)
```

- **γ**：FLL 学习率（越大锁频越快，但噪声敏感）
- **ε = ω - x1**：同相误差
- **x1² + x2²**：幅值平方，用于归一化，防止幅值大时 ωn 被推得太猛
- **ε₀**：小常数（代码中取 1e-6），防止零幅值时除零

同时加上频率范围夹紧，防止异常输入导致发散：

```
ωn = clamp(ωn, 2π·0.3, 2π·3.5)     即 0.3 ~ 3.5 Hz
```

**直觉理解**：当 SOGI 的中心频率偏低时，x2（正交分量）和 ε（误差）有系统性的正相关，`ε·x2 > 0`，`-γ·ε·x2 < 0`，于是 ωn 被推高；反之则被推低。最终 ωn 收敛到输入主频。

## 4. 离散化实现（前向欧拉）

在嵌入式 MCU 或实时控制循环中，用前向欧拉法把连续微分方程离散化。一个控制周期 dt = 1/fs：

```python
class SOGI_FLL:
    def __init__(self, f0_hz, fs, k=1.41, gamma=40.0):
        self.wn = 2π * f0_hz        # 初始中心角频率
        self.k  = k                  # SOGI 阻尼
        self.gamma = gamma           # FLL 学习率
        self.dt = 1.0 / fs           # 采样间隔
        self.x1 = 0.0                # 同相状态
        self.x2 = 0.0                # 正交状态

    def step(self, omega):
        # ---- SOGI 更新 ----
        err = omega - self.x1                        # 步骤 1: 误差
        dx1 = self.k * self.wn * err - self.wn * self.x2  # 步骤 2: x1 导数
        dx2 = self.wn * self.x1                      # 步骤 3: x2 导数
        self.x1 += dx1 * self.dt                     # 步骤 4: 欧拉积分 x1
        self.x2 += dx2 * self.dt                     # 步骤 5: 欧拉积分 x2

        # ---- FLL 更新 ----
        denom = self.x1**2 + self.x2**2 + 1e-6       # 步骤 6: 归一化因子
        dwn = -self.gamma * err * self.x2 / denom     # 步骤 7: 频率调节量
        self.wn += dwn * self.dt                      # 步骤 8: 欧拉积分 ωn
        self.wn = clamp(self.wn, w_min, w_max)        # 步骤 9: 频率范围保护

        # ---- 输出 ----
        phi = atan2(self.x1, -self.x2)               # 步骤 10: 瞬时相位
        amp = sqrt(self.x1**2 + self.x2**2)          # 步骤 11: 瞬时幅值
        return sin(phi), phi, amp, self.wn
```

**计算量统计**：每步约 10 次乘法 + 2 次加法 + 1 次除法 + 1 次 atan2 + 1 次 sin + 1 次 sqrt。在 STM32F4（168 MHz, FPU）上 < 1 μs。

## 5. 从 SOGI 输出到外骨骼力矩

```
┌─────────┐      ┌──────────┐      ┌─────────────┐      ┌──────────┐
│ IMU/编码器│─ω──>│ SOGI-FLL │─φ──>│ sin(φ+lead) │─────>│ × A_GAIN │──> τ_exo
│ (100Hz)  │      │          │─amp─>│ × gate(amp) │      │ × gate   │
└─────────┘      └──────────┘      └─────────────┘      └──────────┘
```

### 5.1 相位超前补偿

控制链路有延迟（传感器滤波 + 电流环 + 机械惯性），典型 20-50 ms。在 1 Hz 步态下，30 ms ≈ 11°。用相位超前直接补偿：

```python
sin_phi_lead = sin(phi + PHI_LEAD)    # PHI_LEAD 取 10°~40°
```

这是**纯相位法最大的好处**：加超前不需要对 ω 微分（微分会放大高频噪声），只需要在 φ 上加一个常数。

### 5.2 幅值看门狗

站立或停步时 ω ≈ 0，但 SOGI 的相位 φ 仍会空转（内部振荡器不会停），此时如果不加保护会"无中生有"地输出力矩。用 amp 做看门狗：

```python
gate = 1.0 if amp > AMP_MIN else 0.0
```

更平滑的版本：

```python
gate = sigmoid((amp - AMP_MIN) / sigma)    # sigma 控制过渡带宽度
```

### 5.3 只做正功模式（可选）

如果绝对不想在任何瞬间做负功：

```python
tau_exo = A_GAIN * gate * max(sin_phi_lead, 0.0)
```

这等于只在半个周期助力，另半个周期力矩为零。代价是平均功率减半且波形不对称。

## 6. 参数调节指南

### 6.1 SOGI 参数

| 参数 | 典型值 | 作用 | 调大会怎样 | 调小会怎样 |
|---|---|---|---|---|
| **f0** (Hz) | 1.0 | FLL 初始频率 | FLL 需要更长时间锁到低频 | 同理，锁到高频慢 |
| **k** | 1.41 | SOGI 阻尼/带宽 | 带宽↑，锁定快，但噪声进来多 | 带宽↓，更干净，但锁定慢 |
| **γ** (gamma) | 40 | FLL 学习率 | 频率跟踪快但抖动大 | 频率跟踪慢但平滑 |
| **f_min, f_max** | 0.3, 3.5 Hz | FLL 频率范围 | 允许更宽的步频变化 | 更安全，防异常锁频 |

### 6.2 力矩输出参数

| 参数 | 典型值 | 说明 |
|---|---|---|
| **A_GAIN** | 3~8 Nm | 力矩幅值，需要根据受试者调 |
| **AMP_MIN** | 20 deg/s | 低于此判定为静止，不出力 |
| **PHI_LEAD** | 0°~40° | 补偿控制链路延迟 |

### 6.3 调参建议优先级

1. 先**固定 k=1.41, γ=40**，只调 A_GAIN 和 AMP_MIN，确认正常出力
2. 然后加 PHI_LEAD，从 0° 逐步增加到 40°，用 corr 指标（或实际代谢/肌电反馈）找最优
3. 如果快/慢走切换时有相位跳动，再调 γ
4. 最后如果想要更贴合步态波形（非正弦），并联 2 倍频 SOGI

## 7. 实验验证结果（S1_Bowen_walkrun.csv）

数据：6 个速度段（1.7/2.8/4.0/5.6/6.7 km/h + 停顿），共 309 秒，100 Hz。

### 7.1 FLL 频率跟踪

| 速度 tag | FFT 真实主频 | FLL 估计 f_est | 差值 |
|---|---|---|---|
| walk_1.7 | 0.90 Hz | 0.855 Hz | -0.045 |
| walk_2.8 | 0.90 Hz | 0.947 Hz | +0.047 |
| walk_4.0 | 1.00 Hz | 0.959 Hz | -0.041 |
| walk_5.6 | 1.30 Hz | 1.303 Hz | +0.003 |
| walk_6.7 | 1.30 Hz | 1.290 Hz | -0.010 |

FLL 在每个速度段都稳稳锁定到真实步频附近。

### 7.2 同相对齐质量

| 速度 tag | sign_match | corr (cos Δφ) | pos_work_ratio |
|---|---|---|---|
| walk_1.7 | 0.705 | +0.728 | +0.240 |
| walk_2.8 | 0.880 | +0.805 | +0.340 |
| walk_4.0 | 0.921 | +0.840 | +0.384 |
| walk_5.6 | 0.826 | +0.763 | +0.356 |
| walk_6.7 | 0.809 | +0.730 | +0.336 |

- **sign_match**：sin(φ) 和 ω 同号的采样点比例。walk_4.0 达到 92%。
- **corr**：归一化互相关 = cos(平均相位差)。0.84 对应 ~33° 相位差。
- **pos_work_ratio**：正功代理指标，均为正值，说明确实在净做正功。

### 7.3 改进空间

corr 未能达到 0.95+ 的原因：
1. 髋角速度含有**2 次谐波**（步态摆动/支撑期不对称），SOGI 只提取基频
2. SOGI 本身的**群延迟**（~70 ms）带来约 25-37° 相移
3. 改进方向：并联 2 倍频 SOGI + 加 PHI_LEAD 补偿

## 8. 边界条件和注意事项

### 8.1 冷启动

SOGI-FLL 从零状态开始需要 1-2 个步态周期（1-2 秒）才能锁定。在此期间：
- amp 从 0 逐渐上升
- φ 不可信
- **必须用 AMP_MIN 门限压住输出**，等锁定后再放行

建议加 ramp-up：

```python
startup_gain = min(1.0, elapsed_sec / RAMP_SEC)
tau_exo *= startup_gain
```

### 8.2 站立 / 停步

ω ≈ 0 时 SOGI 内部振荡器仍在空转（x1, x2 缓慢衰减但不为零）。**只看 amp，不看 φ**：

```python
if amp < AMP_MIN:
    tau_exo = 0    # 不发力
```

### 8.3 速度突变 / 转身

FLL 跟频有惯性，γ=40 时从 0.9 Hz 跟到 1.3 Hz 大约需要 3-5 个周期。如果需要更快响应，调大 γ（代价是稳态更抖）。

### 8.4 瞬态反号

启动期或步频突变时，sin(φ) 可能短暂与 ω 反号。如果业务上不能容忍任何负功：

```python
tau_exo = A_GAIN * max(sin_phi, 0.0) * gate
```

### 8.5 前向欧拉稳定性

前向欧拉要求 dt < 1/(π·f_max)。对于 f_max=3.5 Hz, fs=100 Hz：dt=10 ms < 1/(π·3.5)=91 ms，满足。如果 fs 降到 10 Hz 以下，需要换成半隐式或 RK2 积分。
